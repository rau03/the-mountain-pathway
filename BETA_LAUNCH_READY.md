## ğŸš¨ Quick Support Guide

If a beta tester reports an issue:

### For Authentication Issues

1. Check they're using correct email/password
2. Ask them to try signing up again
3. Check Supabase auth logs
4. Verify environment variables in Vercel

### For Save/Load Issues

1. Verify they're logged in first
2. Check browser console (F12)
3. Look at Supabase database
4. Verify network requests

### For UI/Layout Issues

1. Ask them which device/browser
2. Try to reproduce on same device
3. Check browser compatibility
4. Look for CSS errors in console

### For Performance Issues

1. Check if they have slow internet
2. Look at Supabase query times
3. Check Vercel function logs
4. Ask about browser/device specs
